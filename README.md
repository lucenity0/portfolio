# me.lucenity.dev

An animated pixel-art **"under construction"** landing page — a monochrome night
scene with a snoozing cat, hand-built and fully self-contained.

- Single file: [`index.html`](index.html) — HTML + CSS + a tiny `<canvas>` render loop.
- No frameworks, no fonts, no trackers, no external requests.
- Pixel scene: crescent moon, twinkling stars, a crane, a striped barrier, a
  traffic cone, and a sleeping cat with a curling tail and rising `Zzz`.
- Respects `prefers-reduced-motion` — renders a clean static frame when motion is off.

Served via **GitHub Pages** at **https://me.lucenity.dev**.

## Local preview

```bash
python3 -m http.server 8000
# open http://localhost:8000
```
